# Гениальное приложение по предсказыванию среднего дохода

Это приложение на Streamlit предсказывает, будет ли доход человека превышать 50K, используя модель GradientBoostingClassifier, обученную на наборе данных Adult.

## Структура репозитория

**.gitignore**

Файл, который указывает Git, какие папки и файлы не нужно отслеживать.


**LICENSE**

Файл с лицензией проекта (по умолчанию MIT).


**README.md**

Вы сейчас это читаете.


**app.py**

Основной файл Streamlit-приложения — именно его запускает сервер.


**model.pkl**

Наша лучшая модель, которая используется для предсказания.


**requirements.txt**

Список библиотек Python, необходимых приложению для работы.
